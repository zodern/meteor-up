Single-command Server Setup

Meteor Up can install all of it's dependencies on the servers and prepare them for deployments.

SSL Support

Meteor up can setup nginx and let's encrypt to autogenerate the certificates. It also supports using your custom certificates.

Application Logs

View logs from the terminal, with log tailing and following supported.

Automated Rollback

Meteor Up verifies that the app sucessfully started, and rolls back to the previous version if it failed.

Consistent Results with Docker

Using docker we know that we are using the same meteor enviornment, same proxy, same everything every time. Once you setup your mup config, you no longer have to worry about will this work on this other server. Docker takes care of that for us.
